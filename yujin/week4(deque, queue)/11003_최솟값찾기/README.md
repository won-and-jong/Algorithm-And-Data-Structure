처음으로 푼 플래티넘 문제, 사실 혼자 풀진 못했다

우선순위 큐 개념과 deque를 이용해서 풀어야 했던 문제였다

최솟값을 찾으면 되는데, 범위가 지정되어 있기 때문에 **값의 크기**, **입력 순서** 이 두가지의 정보를 고려해야 했다

그래서 덱에 이 두가지 정보를 저장해주었다.

**덱의 앞쪽에서는 입력순서가 적절한지, 덱의 뒤쪽에서는 값의 크기가 적절한지**를 확인해주면서 원소를 제거해주었다.

즉, 덱의 앞쪽에는 L의 범위에 들어오는 원소이면서 가장 작은 원소가 들어있는 것이다.

따라서 덱의 가장 맨앞 원소의 값을 결과값으로 출력해주면 되었다.
